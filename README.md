# Fitness App

## Project Overview

This is a fitness tracking application.

The goal is to make an app that is easy to use, and has the features that make people want to use it. The goal is to implement a daily workflow with quick and easy logging.

Core Features:

* User Accounts and Profiles
* Workout templates and logging
* Daily check-ins (weight management)
* Dashboard with weekly trends

## Tech Stack

Backend: ASP.NET Core Web API + EF Core + PostGreSQL
Frontend: React + Tailwind
Auth: JWT or OAuth 2.0
Hosting: TBD

## Data Model

This is a high level overview. Expect the following entities to exist:

* User
* Profile
* Workout Template
* Workout Session
* Cardio Session (subtype of Workout Session)
* Daily Checkin
